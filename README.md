# UtilityHub-API

UtilityHub-API is a backend service for managing and comparing utility offers
(energy, gas, telco), built with Symfony and modern PHP.

## Tech Stack
- PHP 8.4
- Symfony 6
- Doctrine ORM

## Architecture

The project follows a modular structure inspired by Domain-Driven Design (DDD).
Each domain module is organized into:

- Domain: core business logic and entities
- Application: use cases and application services
- Infrastructure: framework and external integrations
